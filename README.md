# Student Result Management System (Java)

A simple console-based Java application to manage student results.
It calculates percentage, assigns grade, and stores the record in a text file.

## Features
- Add student details
- Calculates percentage & grade
- Stores record in file
- Displays saved records

## Technologies Used
- Java (OOP)
- File Handling (I/O)

## How to Run?
1. Compile files using:
   `javac *.java`
2. Run the project:
   `java Main`
